# RPG Combat: TDD vs Habit Hooks

Two solutions to the same [RPG Combat kata](https://github.com/ardalis/kata-catalog/blob/master/katas/RPG%20Combat.md), built side by side with AI at the Samman Society unconference for technical coaches, 2026.

Same kata, two ways of working:

```
lexler/   Lada (lexler) — manual, strict TDD. Discipline held by hand.
devill/   Ivett (devill) — habit hooks enforcing the loop. Discipline encoded in tooling.
```

Each folder keeps its own full commit history — the history is the point. Browse the commits to see how each approach unfolds.

## Branches

- `lexler` — Lada's solution and its history
- `devill` — Ivett's solution and its history
- `main` — both, merged, in their own folders
